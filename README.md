# Todoアプリ (Markdoor 面接前タスク)

シンプルなタスク管理・追跡アプリケーションです。バックエンドにFastAPI、データベースにSQLiteを採用し、フロントエンドは最小限のHTML/JSで実装しています。

## 実装済み機能
* **タスクの追加**: タイトル（必須）と詳細（任意）を入力して新しいタスクを作成できます。
* **タスクリストの表示**: 全タスクを一覧で表示し、完了済みのタスクには取り消し線が引かれます。
* **タスク状態の更新**: チェックボックスで「完了・未完了」の切り替えができます。
* **タスクの削除**: 誤操作防止のため、削除ボタン押下時に確認プロンプトを表示します。
* **データ永続化**: SQLiteを使用してタスクデータを保存します。

## 技術スタック
* **バックエンド**: Python, FastAPI, SQLAlchemy
* **データベース**: SQLite
* **フロントエンド**: HTML, CSS, JavaScript (Vanilla)

## セットアップ手順

### 1. 必要なライブラリのインストール
ターミナルで以下のコマンドを実行し、FastAPIなどの必要なパッケージをインストールします。
\`\`\`bash
python -m pip install fastapi uvicorn sqlalchemy pydantic
\`\`\`

### 2. バックエンドサーバーの起動
同ディレクトリ内で以下のコマンドを実行し、ローカルサーバーを立ち上げます。
\`\`\`bash
python -m uvicorn main:app --reload
\`\`\`
※初回起動時に、同ディレクトリ内にデータベースファイル（`todos.db`）が自動生成されます。

### 3. フロントエンドの起動
サーバーを起動したままの状態で、プロジェクトフォルダ内にある `index.html` をGoogle ChromeなどのWebブラウザにドラッグ＆ドロップして開いてください。
そこからアプリを操作できます。
