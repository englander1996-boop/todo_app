# Todoアプリ

シンプルなタスク管理・追跡アプリケーションです。バックエンドにFastAPI、データベースにSQLiteを採用し、フロントエンドは最小限のHTML/JSで直感的なUXを提供できるよう実装しています。

## 実装済み機能（要件定義への対応）

### 基本機能
* **タスクの追加**: タイトル（必須）、詳細（任意）、期限を入力してタスクを作成。作成日時は自動付与されます。
* **タスクの編集**: 専用のモーダルウィンドウで、タスクのタイトルと詳細をスムーズに更新できます。
* **タスク状態の更新**: チェックボックスでの「完了/未完了」切り替えに加え、重要タスクのマーク（★/☆）も可能です。
* **タスクの削除**: 誤操作防止のため、削除ボタン押下時に確認プロンプトを表示します。
* **タスクの表示・操作**: 
  * 完了済みタスクは取り消し線で視覚的に区別。
  * **絞り込み**: 全タスク、未完了のみ、完了済みのみでフィルタリング可能。
  * **並び替え**: 作成日時の新しい順、古い順でソート可能。
* **ユーザーフィードバック**: タスクの追加、編集、削除、状態変更時に画面下部へトースト通知を表示します。

### データ管理（オプション要件対応）
* **データ永続化**: SQLiteを使用してタスクデータを保存します。
* **エクスポート / インポート機能**: 
  * 登録されている全タスクをJSON形式でダウンロード（エクスポート）できます。
  * JSONファイルを読み込み、バックエンドの一括登録APIを通じてデータを復元（インポート）できます。

## 技術スタック
* **バックエンド**: Python, FastAPI, SQLAlchemy
* **データベース**: SQLite
* **フロントエンド**: HTML, CSS, JavaScript (Vanilla)
* **インフラ（オプション要件対応）**: Docker, Docker Compose

## セットアップ手順

本アプリは、Dockerを使用した簡単な起動（推奨）と、ローカル環境での起動の2通りをサポートしています。

### Option A: Dockerを使用する場合（推奨）
Docker Desktop等のコンテナ環境がインストールされている場合は、以下のコマンド一発で起動します。

1. ターミナルでプロジェクトのルートディレクトリを開きます。
2. 以下のコマンドを実行します。
```bash
docker compose up -d
